
# Tiny2D 物理シミュレーション・ドローイングゲーム

`Tiny2D.js` を使用した、ブラウザ上で動くシンプルな2D物理シミュレーションゲームです。画面上にマウスで線を引くことができ、上から落ちてくるボールをバウンドさせたり、軌道を変えたりして遊ぶことができます。
<img width="505" height="356" alt="スクリーンショット 2026-06-08 064246" src="https://github.com/user-attachments/assets/ac86e64f-c015-4923-a4ec-d7d069ef11a2" />

本プロジェクトは Node.js (Express) を用いた静的ウェブサーバーとして構成されています。

## 🚀 機能概要

- **自動ボール生成**: 一定時間（5秒）ごとに、画面上部からランダムな位置に青いボールが落下します。
- **自由な線画**: キャンバス上をドラッグ＆ドロップすることで、物理判定を持った固定の「壁（線）」を配置できます。
- **カラーパレット**: 画面上部のパレットから、次に描く線の色（全8色）を選択できます。
- **本格的な物理演算**: 重力加速度や衝突判定がリアルタイムに計算されます。

🛠️ 動作環境
Node.js: v14 以上の環境を推奨

ブラウザ: Google Chrome / Microsoft Edge / Safari などの主要なモダンブラウザ

💻 セットアップと起動方法
1. 依存関係のインストール
プロジェクトのルートディレクトリで以下のコマンドを実行し、必要なパッケージ（Express）をインストールします。

Bash
npm install
※ package.json がない状態で手動構築する場合は、先に npm init -y を実行してから npm install express を行ってください。

2. サーバーの起動
以下のコマンドでローカルサーバーを立ち上げます。

Bash
node app.js
3. ゲームへのアクセス
サーバーが起動したら、ブラウザを開き次のURLにアクセスしてください。
👉 http://localhost:3000

🎮 遊び方
画面上部のパレットから好きな色を選択します。

画面（木目のキャンバス）の上でマウスを左クリックしながらドラッグします。

クリックを離すとそこに「線」が固定されます。

上から落ちてくる青いボールが、描いた線に当たってリアルに弾む様子を観察しましょう！
